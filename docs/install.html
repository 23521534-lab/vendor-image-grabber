INSTALL

The extension loads as an unpacked extension in Chrome. No Chrome Web Store required — takes about 2 minutes.

⬇ Download ZIP from GitHub View on GitHub ↗
1

Download the extension files

Click Download ZIP from GitHub above, or go to github.com/23521534-lab/vendor-image-grabber → click the green Code button → Download ZIP. Once downloaded, unzip the file on your computer.

2

Open Chrome Extensions

In Chrome, navigate to chrome://extensions in the address bar and press Enter.

3

Turn on Developer Mode

In the top-right corner of the Extensions page, toggle Developer mode on. This unlocks the ability to load local extensions.

4

Load the unpacked extension

Click Load unpacked (top-left). In the file dialog, navigate to and select the unzipped vendor-image-grabber-main/ folder. Click Select/Open.

5

Pin the extension to your toolbar

Click the puzzle piece 🧩 icon in Chrome's toolbar → find Vendor Image Grabber → click the pin icon. The extension icon now appears permanently in your toolbar.

6

Navigate to a vendor page and scan

Go to bassettmirror.com, worldwidehomefurnishingsinc.com, or leisuremod.com — browse to a product page, click the extension icon, and hit Scan for Images.

Extension folder

What You're Loading

Chrome loads the entire folder as the extension. Here's what each file does.

vendor-image-grabber/
vendors.js    ← Add new vendors here
manifest.json   ← Extension config & permissions
popup.html     ← Extension popup UI
popup.js       ← UI engine & download logic
background.js   ← Service worker (download events)
icons/         ← Extension icons (16, 48, 128px)
Permissions

What It Accesses

The extension requests only the permissions it needs.

activeTab

Read the URL of the current tab to detect which vendor you're on. No access to other tabs.

scripting

Inject the scraper function into the active page. Required to run JavaScript that scans the page's DOM for image URLs.

downloads

Save files to your Downloads folder using Chrome's native download manager. Images go into a named subfolder.

storage

Persist settings (auto-select, throttle, dedup) across browser sessions using Chrome's local storage API.

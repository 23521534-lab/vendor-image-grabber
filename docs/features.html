FEATURES

Everything built into the extension — from detection methods to download management to filtering.

🧠

Multi-Method Image Detection

Each vendor runs 4 parallel detection methods on the active page. Finds images that lazy-load, that are hidden in data attributes, or embedded in raw HTML as CDN URLs.

img[src] data-zoom-image background-image CDN regex
🖼

Live Thumbnail Preview

Before you download anything, every found image renders as a thumbnail inside the extension popup. Click to toggle, click "Select All" to grab everything. Images that fail to load show a fallback placeholder.

🔎

Format Filter

Filter the image grid by file format — All, JPG, PNG, or WEBP. Useful when a page mixes product photos (JPG) with transparent PNGs for icons or overlays you don't want.

All JPG PNG WEBP
⬇

Throttled Batch Download

Downloads fire with a configurable 280ms delay between each file. This prevents the browser from queueing too many simultaneous downloads and avoids network timeouts on slower connections.

280ms throttle Async queue
📁

Organized Subfolders

Images save to a vendor-specific subfolder inside ~/Downloads/vendor-images/. Each vendor gets its own folder so files from different sites never mix.

vendor-images/bassettmirror-com/
🏷

Original Filenames Preserved

Files save using the exact filename from the server URL (e.g. BA-6398LNS-A.jpg). Duplicate filenames are automatically renamed to avoid overwrites, configurable in settings.

⚙

Settings Panel

Persistent settings stored with chrome.storage. Toggle auto-select, download throttle, and dedup behavior. Settings persist across browser sessions.

📋

Live Activity Log

An inline console inside the popup shows real-time download status per file — green for success, red for failures. Scrollable so you can review everything after a large batch.

✓ Saved: filename.jpg ✕ Failed: ...

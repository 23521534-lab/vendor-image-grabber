VENDORS

Each vendor has a custom-written scraper tuned to its specific HTML structure and image hosting setup.

🪞
Bassett Mirror
bassettmirror.com ↗
ACTIVE

Furniture and mirror catalog. Product images are served through multiple mechanisms — standard img tags, zoom gallery data attributes, and CSS background images on swatch elements.

img[src] data-zoom-image data-large / data-full background-image CSS a[href] image links CDN HTML scan
Best pages to scan: Individual product pages and category listing pages. Zoom images give the highest resolution. The scraper automatically filters icons, logos, and UI sprites.
🏠
Worldwide Home
worldwidehomefurnishingsinc.com ↗
ACTIVE

Wholesale home furnishings vendor. Images are served from a separate portal CDN subdomain. Product pages also contain a DOWNLOADS section with direct full-resolution image URLs listed as plain text.

portal CDN regex DOWNLOADS section parser img[src] a[href] image links
Best pages to scan: Individual product pages — the DOWNLOADS section contains the highest-resolution images. The portal CDN pattern catches all images embedded in the page HTML.
